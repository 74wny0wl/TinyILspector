# TinyILspector
Tiny tool that inpects MS IL in .NET Assembly

```
Description:
  Inspects MS IL in .NET Assembly

Usage:
  TinyILspector [options]

Options:
  -i, --input <input>                     The path to the executable that is to be inspected.
  -mt, --metadata-token <metadata-token>  Requested metadata token
  -fn, --function-name <function-name>    Requested function name
  --version                               Show version information
  -?, -h, --help                          Show help and usage information
  ```
